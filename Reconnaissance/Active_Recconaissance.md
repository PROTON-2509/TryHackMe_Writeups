# Active Reconnaissance – TryHackMe

## Objective
The objective of this room was to understand what active reconnaissance is and how it is used to gather information by directly interacting with a target system.

## Environment
- Platform: TryHackMe
- Tools Used: Nmap, ping, traceroute
- Operating System: Linux
- Access Method: Web-based terminal / AttackBox

## Concepts & Skills Practiced
- What active reconnaissance is
- Difference between active and passive reconnaissance
- Host discovery
- Port scanning
- Service detection
- Identifying open ports and running services
