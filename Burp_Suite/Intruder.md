# Burp Suite: Intruder – TryHackMe

## Objective
The objective of this room was to understand how Burp Suite Intruder works and how it is used to automate attacks by sending multiple modified HTTP requests.

## Environment
- Platform: TryHackMe
- Tool: Burp Suite (Intruder)
- Operating System: Any
- Access Method: Web-based learning room / AttackBox

## Concepts & Skills Practiced
- Sending requests to Intruder
- Setting attack positions
- Understanding attack types
- Using payload lists
- Analyzing response length and status codes
- Identifying potential vulnerabilities
