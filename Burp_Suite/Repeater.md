# Burp Suite: Repeater – TryHackMe

## Objective
The objective of this room was to understand how to use Burp Suite Repeater to manually modify and resend HTTP requests for testing web applications.

## Environment
- Platform: TryHackMe
- Tool: Burp Suite (Repeater)
- Operating System: Any
- Access Method: Web-based learning room / AttackBox

## Concepts & Skills Practiced
- Sending requests from Proxy to Repeater
- Modifying HTTP request parameters
- Resending and analyzing responses
- Testing input fields manually
- Understanding server responses
