# Nmap: Basic Port Scans – TryHackMe

## Objective
The objective of this room was to understand how Nmap performs basic port scanning to identify open ports and services on a target system.

## Environment
- Platform: TryHackMe
- Tool: Nmap
- Operating System: Linux
- Access Method: Web-based terminal / AttackBox

## Concepts & Skills Practiced
- What port scanning is
- Scanning a single host
- Scanning multiple ports
- Scanning all ports
- Understanding open, closed, and filtered ports
- Reading and interpreting scan results
