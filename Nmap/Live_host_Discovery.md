# Nmap: Live Host Discovery – TryHackMe

## Objective
The objective of this room was to understand how Nmap performs host discovery and identifies live systems on a network.

## Environment
- Platform: TryHackMe
- Tool: Nmap
- Operating System: Linux
- Access Method: Web-based terminal / AttackBox

## Concepts & Skills Practiced
- What host discovery is
- Identifying live hosts on a network
- ICMP echo requests
- TCP SYN ping
- TCP ACK ping
- Disabling ping scans
- Understanding scan results
